TP Gestion de contact

Ce TP consiste en la realisation d'un programme de gestion de contacts.

 

Preparation des outils

·         Copier le contenu du fichier d05-e21-juillet-groupe1-script-db.sql.

·         Ouvrir Sql Server Management Studio et cliquer sur nouvelle requete.

·         Coller le contenu.

·         Lancer la requete pour creer la base de donnees avec toutes les tables.

·         Ajouter une ligne dans la table Usagers, que vous utiliserez pour vous authentifier sur l'application. Mettre 2 dans la colonne role pour etre gestionnaire, 3 pour administrateur et 1 pour un simple utilisateur.

·         Changer la chaine de connexion dans le fichier App.config de la couche Vue, pour ajouter le nom de votre server.

Vous pouvez maintenant lancer le programme.

Utilisation

Login

Vous devez vous authentifier pour acceder aux fonctionalites. Saisir l'identifiant et le mot de passe saisis dans la table Usagers.

Ajouter un Contact

Saisir les information sur le contact, sachant que le couple nom et prenom doit etre unique, l'email et le telephone doivent etre uniques. Le telephone doit etre dans le format (000)000-0000.

Rechercher des Contacts

Choisir le critere de recherche dans le combobox puis saisir la valeur dans le textbox. Avant de faire une operation sur un contact (Modifier, supprimer ou ajouter aux favoris), vous devez selectionner la ligne correspondant a ce contact.

Envoyer un message

Saisir le prenom du contact dans le champ de recherche. Puis saisir le contenu du message a envoyer.

Gestions des usagers

Si vous avez les droits Administrateur ou Gestionnaire, vous aurez acces a la gestion des usagers.

Ajouter des usagers

Cliquer sur le bouton ajouter pour ajouter un nouvel usager. Puis saisir les informations, sachant que l'identifiant doit etre unique.
